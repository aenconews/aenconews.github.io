Newly Listed Trading Pairs on AENX

We are excited to announce that we have added trading pairs with Uniswap (UNI), Theta Token (THETA) and the Graph (GRT) on AENX!

The newly added trading pairs are:
  • UNI/BTC, UNI /USDT
  • THETA/BTC, THETA/ETH, THETA/USDT
  • GRT/BTC, GRT/ETH, GRT/USDT

More about Uniswap (UNI):
Uniswap (UNI) behaves as an Ethereum token of the Uniswap network. The Uniswap network is an automated liquidity provider designed for easy ERC20 token exchange. UNI tokens are exchanged via smart-contract-defined liquidity pools. It is a popular decentralized trading protocol, known for its role in facilitating automated trading of decentralized finance (DeFi) tokens.

Learn more: https://uniswap.org

More about Theta (THETA):
Theta Token (THETA) behaves as a governance token of the Theta network. They provide a series of functionalities to their users. The Theta mainnet operates as a decentralized network in which users share bandwidth and computing resources on a peer-to-peer (P2P) basis. You can use the token to stake as a Validator or Guardian node, in order to produce blocks and participate in protocol’s governance.

Learn more: https://www.thetatoken.org

More about the Graph (GRT)
Graph token(GRT) behaves as an Ethereum token of the Graph network. The Graph network is the marketplace of service providers, which ensures the transparency of data and optimal operation of dApps and APIs built on the indexing protocol. It is an indexing protocol for querying data for networks like Ethereum and IPFS, powering many applications in both DeFi and the broader Web3 ecosystem. Anyone can build and publish open APIs, called subgraphs, that applications can query using GraphQL to retrieve blockchain data.

Learn more: https://thegraph.com
